Insert custom messages into comment notification emails.

== Description ==

Notiflyer – Notice Flyer. Ideal for large websites with multiple authors and contributors where UGC (user generated content) is accepted. Notiflyer allows you to add a custom message to be added to the notification email sent out in an email to the post’s author when the post receives a comment.

The custom message can contain links, such as links to your social media accounts, or it can contain alerts for offers, discounts, product updates, events or anything you would like to add. 

Notifyler is particularly useful for attracting people towards your social channels on multi-author sites.

== Installation ==

1. Upload `notiflyer` folder to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress

== Frequently Asked Questions ==

== Can I include HTML in the notification?

No, HTML cannot be included in the notification.

== Can I include links in the notification?

Yes, links can be included in the notification emails.
